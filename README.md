#  JavaScript Learning Journey

A complete collection of my **JavaScript learning practice** — from basics to advanced concepts.  
All code examples and notes are inspired by [W3Schools JavaScript Tutorial](https://www.w3schools.com/js/).  
This repository documents my **daily progress** and **hands-on experiments** while mastering core JavaScript.

---

##  Topics Covered

###  Fundamentals
- Variables and Data Types  
- Operators and Expressions  
- Conditional Statements (if, else, switch)  
- Loops (for, while, do-while, break, continue)

###  Strings
- String Methods (`slice()`, `substring()`, `substr()`)  
- Template Literals  
- String Search, Replace, and Case Conversion  
- Working with `charAt()`, `charCodeAt()`, `codePointAt()`  
- Real-world string formatting examples  

###  DOM Interaction
- Using `getElementById()`  
- Manipulating `innerHTML`  
- Event handling with `onclick`
